# 画面設計

## 画面一覧

| # | 画面名 | パス | 権限 | 説明 |

| S-001 | ログイン | /login | 未認証 | メール+パスワードでログイン |

| S-015 | 通知一覧 | /notifications | 全ロール | 通知の確認・既読化 |

## コンポーネント一覧

| コンポーネント | 用途 |
|--------------|------|
| Layout | 共通レイアウト（ヘッダー + サイドバー + コンテンツ） |
| Header | ロゴ、ナビ、通知ベル、ユーザーメニュー |
| Sidebar | ナビゲーションメニュー（ロール別表示制御） |
| NotificationBell | 未読通知数バッジ付きベルアイコン |
| TimeRecordButton | 打刻ボタン（状態に応じて活性/非活性） |
| LocationInfo | 現在地・最寄り拠点表示 |
| AttendanceTable | 月次勤怠テーブル |
| ApplicationForm | 各種申請フォーム（休暇/残業/打刻修正） |
| ApprovalCard | 承認依頼カード（承認/差戻ボタン付き） |
| StatusBadge | ステータス表示バッジ |
| Pagination | ページネーション |
| ConfirmDialog | 確認ダイアログ |
| Toast | トースト通知 |
